# HR Analytics – Compensation & Benefits Dashboard

Інтерактивний дашборд для аналізу зарплат, бонусів та пільг співробітників із можливістю порівняння внутрішніх даних із ринковими.

---

## 🎯 Мета проекту
- Надати HR‑відділу інструмент для швидкого аналізу компенсацій.
- Виявити дисбаланс між внутрішніми зарплатами та ринковими.
- Підтримати прийняття рішень щодо бонусів і пільг.

---

## ⚙️ Технології
- **Python** (pandas, numpy, seaborn, matplotlib, plotly, streamlit)
- **Streamlit** для веб‑інтерфейсу
- **JSON** для зберігання ринкових зарплат
- **CSV** як джерело даних
